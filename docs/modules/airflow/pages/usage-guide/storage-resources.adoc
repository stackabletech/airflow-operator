= Resource Requests

include::home:concepts:stackable_resource_requests.adoc[]

If no resource requests are configured explicitly, the operator uses the following defaults:

[source,yaml]
----
spec:
  schedulers:
    config:
      resources:
        cpu:
          min: 100m
          max: 400m
        memory:
          limit: 512Mi
  workers:
    config:
      resources:
        cpu:
          min: 200m
          max: 800m
        memory:
          limit: 1750Mi
  webservers:
    config:
      resources:
        cpu:
          min: 100m
          max: 400m
        memory:
          limit: 700Mi
----
